
AJAX EXAMPLE, LOCATIONS FROM CSV TABLE

This example loads marker locations from a CSV table via ajax. You can easily rewrite this example to load data in XML or JSON format.


1.	Upload this directory to a web server.

2.	You can include('myearth.htm') in your website or call the file directly. 

3.	Add your data to locations.csv add the columns you need, e.g. for links, colors, ...

4. 	Modify myearth.htm and add the properties, behavior and styles you need.
