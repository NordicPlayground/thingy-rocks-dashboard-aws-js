
PHP EXAMPLE, LOCATIONS FROM MYSQL DATABASE

This example loads marker locations from a database.


1.	Import marker_locations.sql into your database.

2.	Setup your database connection in myearth.php

3.	Upload myearth.php to a web server.

4.	You can include('myearth.php') in your website or call the file directly. 

5.	Add the database fields you need, e.g. for links, colors, ...

6. 	Modify myearth.php and add the properties, behavior and styles you need.
